Tennis Rules

  • A match is played in best-of-3 or 5 sets.
  • A set is won by winning 6 games with at least 2 game lead.
  • Each game uses the scoring: 0, 15, 30, 40, game.
  • If tied at 40–40, it's called "deuce", then advantage rules apply.
